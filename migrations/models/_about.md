# Модели для Миграций

### Описание
Эта директория содержит "слепки" (snapshots) моделей `Tortoise ORM` на момент генерации каждой миграции.

### Назначение
- **История схемы:** `aerich` использует эти файлы для сравнения версий схемы базы данных. Когда вы создаете новую миграцию, `aerich` сравнивает текущее состояние ваших моделей в `durak/db/models` с последней сохраненной версией в этой директории.
- **Основа для SQL:** На основе разницы между этими "слепками" и генерируется SQL-код для обновления базы данных.

### Взаимосвязи
- **Управляется:** Исключительно инструментом `aerich`. Ручное редактирование файлов в этой директории не предполагается.
