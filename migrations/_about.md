# Миграции Базы Данных

### Описание
Эта директория содержит файлы миграций, сгенерированные инструментом `aerich`. Миграции представляют собой упорядоченные изменения схемы базы данных, что позволяет безопасно обновлять структуру БД по мере развития проекта.

### Назначение
- **Контроль версий схемы БД:** Позволяет отслеживать и применять изменения в моделях данных (`durak/db/models`) к рабочей базе данных (PostgreSQL).
- **Автоматизация:** `aerich` автоматически сравнивает текущие модели с состоянием базы данных и генерирует необходимые SQL-скрипты для их синхронизации.

### Взаимосвязи
- **Связана с:**
    - `pyproject.toml`: В этом файле настраивается `aerich`, указывается путь к моделям и подключение к БД.
    - `durak/db/models/`: `aerich` отслеживает изменения в моделях, определенных в этой директории.
