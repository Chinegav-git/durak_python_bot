# Changelog

## [1.12.15] - 2024-07-31

### Bug Fixes

- Исправлена критическая ошибка `TelegramBadRequest: DOCUMENT_INVALID`, возникавшая в инлайн-режиме.
- **Причина:** В функции `game_info` (`durak/logic/result.py`) использовалось Markdown-форматирование (обратные кавычки `` ` ``) для отображения карт на столе, в то время как для всего сообщения был установлен `parse_mode="HTML"`.
- **Решение:** Заменены Markdown-кавычки на соответствующие HTML-теги `<code>`, что устранило конфликт форматирования.
