# Изображения Карт (Высокое Разрешение)

### Описание
Эта директория содержит отдельные PNG-файлы для каждой игральной карты в высоком разрешении (512px).

### Назначение
- **Основа для спрайтов:** Эти изображения являются исходным материалом для создания объединенного спрайт-листа `sprites.png`.
- **Архив:** Служат в качестве эталонного набора графики для темы карт "Classic".

### Взаимосвязи
- **Не используются напрямую** в коде бота. Вместо них используются `file_id` стикеров, полученные из этих изображений.
