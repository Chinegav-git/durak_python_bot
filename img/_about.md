# Графические Ассеты

### Описание
Эта директория является корневым хранилищем для всех графических ассетов, используемых в боте. Изображения напрямую не встраиваются в код; вместо этого их `file_id` кешируются в `durak/objects/card.py` для быстрой отправки в виде стикеров.

### Содержимое
- **Спрайт-листы:** `sprites.png`, `sprites_lowsat.png` — файлы, объединяющие изображения всех карт.
- **Логотип:** `logo_sticker_pack.png` — основное изображение для стикерпака бота.
- **Вложенные директории:**
    - `sprite_512px/`: Отдельные изображения карт в высоком разрешении.
    - `sprites_512px_lowsat/`: Аналогично предыдущей, но с пониженной насыщенностью.
    - `special/`: Изображения для специальных игровых действий ("Взять", "Пас").
