# Изображения Карт (Низкая Насыщенность)

### Описание
Эта директория содержит отдельные PNG-файлы для каждой игральной карты в высоком разрешении (512px), но с пониженной насыщенностью цветов.

### Назначение
- **Визуальное состояние:** Эти "серые" версии карт используются для отображения карт, которые уже были отыграны в текущем кону, но еще остаются на столе. Это помогает игрокам лучше ориентироваться в текущей игровой ситуации.
- **Основа для спрайтов:** Являются исходным материалом для создания объединенного спрайт-листа `sprites_lowsat.png`.

### Взаимосвязи
- **Не используются напрямую** в коде бота. Вместо них используются `file_id` стикеров, полученные из этих изображений.
