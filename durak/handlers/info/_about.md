# Информационные Обработчики

### Описание
Эта директория содержит обработчики `aiogram`, предоставляющие пользователям справочную информацию и инструменты, не связанные напрямую с игровым процессом.

### Назначение
- **Информирование пользователей:** Обработчики отвечают на такие команды, как `/help` и `/stats`, предоставляя пользователям руководства по игре и их личную статистику.
- **Инструменты для разработки:** Здесь также могут располагаться вспомогательные обработчики для администраторов, например, `get_sticker_id.py` для получения `file_id` стикеров.

### Взаимосвязи
- **Вызывается из:** `bot.py` (через `durak/handlers/__init__.py`), который регистрирует роутеры из этой директории.
- **Использует:**
    - `durak/db/`: Для доступа к статистике пользователей (`UserSetting`).
    - `durak/handlers/settings.py`: Для перенаправления пользователей в меню настроек за более детальной информацией.
