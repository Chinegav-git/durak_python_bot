# Описание папки durak

Это основной пакет, содержащий всю бизнес-логику игры "Дурак".

## Содержимое

- `db/`: Модули для работы с базой данных.
- `handlers/`: Обработчики команд и callback-запросов от пользователей в Telegram.
- `logic/`: Ключевая бизнес-логика игры.
- `objects/`: Классы, описывающие основные сущности (модели данных) игры.
- `filters.py`: Кастомные фильтры для `aiogram`.
- `_about.md`: Этот файл.

# durak Folder Description

This is the main package containing all the business logic of the "Durak" game.

## Contents

- `db/`: Modules for working with the database.
- `handlers/`: Handlers for commands and callback queries from users in Telegram.
- `logic/`: Key business logic of the game.
- `objects/`: Classes describing the main game entities (data models).
- `filters.py`: Custom filters for `aiogram`.
- `_about.md`: This file.
