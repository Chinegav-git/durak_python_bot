# Описание папки db

Этот пакет отвечает за взаимодействие с базой данных.
Он содержит модели данных, конфигурацию подключения и утилиты для работы с БД.

## Содержимое

- `models.py`: Определяет модели `User` и `Chat` для Tortoise ORM.
- `tortoise_config.py`: Конфигурация подключения к базе данных Tortoise ORM.
- `user_settings.py`: (устарел, используется `models.py`) Определял модель `UserSetting`.
- `__init__.py`: Начальный файл пакета.
- `_about.md`: Этот файл.

# db Folder Description

This package is responsible for interacting with the database.
It contains data models, connection configuration, and database utilities.

## Contents

- `models.py`: Defines the `User` and `Chat` models for Tortoise ORM.
- `tortoise_config.py`: Tortoise ORM database connection configuration.
- `user_settings.py`: (deprecated, `models.py` is used) Defined the `UserSetting` model.
- `__init__.py`: Package initialization file.
- `_about.md`: This file.
