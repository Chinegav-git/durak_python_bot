# Темы Карт

### Описание
Эта директория хранит различные темы оформления для игральных карт. Каждая тема определяет, какой `file_id` стикера будет использоваться для отображения каждой карты в разных состояниях.

### Назначение
- **Персонализация:** Позволяет пользователям выбирать визуальный стиль карт через команду `/settings`, делая игровой опыт более разнообразным.
- **Расширяемость:** Система спроектирована так, чтобы можно было легко добавлять новые темы. Достаточно создать новый `.py` файл с определенным словарем `THEME`, и он будет автоматически подхвачен `ThemeManager`'ом.

### Структура файла темы
Каждый файл в этой директории (например, `classic.py`) должен содержать словарь `THEME` со следующей структурой:
```python
THEME = {
    'normal': {'6_d': 'file_id_1', ...},
    'grey': {'6_d': 'file_id_2', ...},
    'trump_normal': {'6_d': 'file_id_3', ...},
    'trump_grey': {'6_d': 'file_id_4', ...}
}
```

### Взаимосвязи
- **Используется:**
    - `durak/objects/card.py` (в частности, `ThemeManager`): Динамически загружает все файлы тем из этой директории для предоставления `file_id` по запросу.
