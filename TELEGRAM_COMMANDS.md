# Команди для керування ботом Дурак

## Базові команди для гравців

### Головне меню
- `/start` - Запуск бота та показ головного меню
- `/help` - Показати довідку та список команд

### Управління лобі
- `/new` - Створити нову гру
- `/join` - Приєднатися до існуючої гри
- `/leave` - Покинути поточну гру
- `/start` - Почати гру (тільки для власника лобі)

### Ігрові дії
- `/attack` - Атакувати (покласти карту)
- `/defend` - Захищатися (покласти карту для захисту)
- `/take` - Взяти карти (коли не можна захиститися)
- `/done` - Закінчити хід

### Інформаційні команди
- `/info` - Показати інформацію про поточну гру
- `/players` - Показати список гравців у лобі
- `/cards` - Показати ваші карти

### Налаштування
- `/settings` - Відкрити меню налаштувань
- `/theme` - Змінити тему карт
- `/mode` - Змінити режим гри
- `/language` - Змінити мову інтерфейсу

## Адміністративні команди

### Управління грою
- `/kick [username]` - Видалити гравця з гри
- `/close` - Закрити поточну гру
- `/kill [game_id]` - Примусово завершити гру

### Системні команди
- `/status` - Показати статус бота
- `/stats` - Показати статистіку сервера

## Команди для налагодження (тільки для адміністраторів)

- `/debug` - Включити режим налагодження
- `/test` - Тестові команди
- `/reload` - Перезавантажити конфігурацію

## Як користуватися

1. **Створення гри**: `/new` → чекаєте гравців → `/start`
2. **Приєднання**: `/join` → чекаєте початку гри
3. **Хід гри**: `/attack` → опонент `/defend` або `/take` → `/done`

## Правила гри

- Мета: позбутися всіх карт
- Козир: визначається автоматично
- Перший хід: у гравця з найменшою козирною картою
- Кількість гравців: 2-6

## Приклад діалогу

```
Ви: /new
Бот: Створено нову гру. ID: #12345

Ви: /join
Бот: Ви приєдналися до гри #12345

Ви: /start
Бот: Гра почалася! Ваші карти: [6♣, 7♥, K♦]

Ви: /attack 6♣
Бот: Ви атакували картою 6♣

Опонент: /defend 7♣
Бот: Захист успішний!

Ви: /done
Бот: Ваш хід завершено
```

## Примітки

- Всі команди доступні через меню кнопок
- Можна використовувати текстові команди
- Бот автоматично підказує доступні дії
- Ігровий процес збережається навіть при перезавантаженні
