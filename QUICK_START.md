# 🚀 Быстрый запуск бота "Дурак"

## ✅ Что уже настроено:
- PostgreSQL 13.15 (порт 5432)
- Redis (порт 6379)
- Виртуальное окружение Python
- Все зависимости установлены
- Таблицы в базе данных созданы
- **Исправлена архитектура Bot API** (aiogram 3.x совместимость)
- **Исправлены контекстные менеджеры** БД
- **Автоматизированные скрипты** запуска/остановки

## 🎯 Быстрый запуск:

### Вариант 1: Автоматический запуск (рекомендуется)
```bash
./start_bot.sh
```
Скрипт автоматически проверит и запустит все необходимые сервисы.

### Вариант 2: Ручной запуск
```bash
# 1. Запуск Redis (если не запущен)
redis-server --daemonize yes

# 2. Запуск PostgreSQL (если не запущен)
sudo -u postgres sh -c "cd /tmp && /usr/local/pgsql/bin/pg_ctl -D /var/lib/postgresql/data -l /tmp/postgres.log start"

# 3. Активация виртуального окружения и запуск бота
source venv/bin/activate
python bot.py
```

## 🛑 Остановка сервисов:
```bash
./stop_all.sh
```

## 📊 Проверка статуса сервисов:
```bash
# Redis
redis-cli ping
# Должно ответить: PONG

# PostgreSQL
sudo -u postgres sh -c "cd /tmp && /usr/local/pgsql/bin/pg_ctl -D /var/lib/postgresql/data status"
```

## 🔧 Конфигурация:
- **Токен бота**: в файле `.env`
- **База данных**: PostgreSQL `durak_db` с пользователем `chinegav`
- **Redis**: localhost:6379

## 📝 Логи:
- PostgreSQL: `/tmp/postgres.log`
- Redis: вывод в консоль при запуске
- Бот: вывод в консоль

## 🎮 Управление ботом:
Основные команды в Telegram:
- `/new` - создать новую игру
- `/join` - присоединиться к игре
- `/start` - начать игру
- `/leave` - покинуть игру
- `/help` - помощь

Бот готов к использованию! 🎉
