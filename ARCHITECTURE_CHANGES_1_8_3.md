## 📋 Сводка изменений в структуре проекта (v1.8.3)

## 🔄 Ключевые изменения архитектуры

### 1. Исправлена работа с Bot API (aiogram 3.x)
**Проблема:** `Bot.get_current()` не существует в aiogram 3.x
**Решение:** Передача `bot` экземпляра через параметры функций

**Затронутые файлы:**
- `durak/logic/actions.py` - 9 функций исправлены
- `durak/handlers/game/actions.py` - 2 вызова обновлены
- `durak/handlers/game/leave.py` - 1 вызов обновлен
- `durak/handlers/game/kick.py` - 1 вызов обновлен
- `durak/handlers/game/auto_leave.py` - 1 вызов обновлен
- `durak/handlers/game/global_leave.py` - 1 вызов обновлен
- `durak/handlers/game/lobby_kick.py` - 1 вызов обновлен

### 2. Исправлены контекстные менеджеры БД
**Проблема:** `'coroutine' object does not support the asynchronous context manager protocol`
**Решение:** Переработан `get_db_session()` в `tortoise_config.py`

**Затронутые файлы:**
- `durak/db/tortoise_config.py` - класс `DatabaseSession` исправлен

### 3. Добавлены скрипты автоматизации
**Нововведение:** Удобные скрипты для управления ботом

**Новые файлы:**
- `start_bot.sh` - автоматический запуск всех сервисов
- `stop_all.sh` - остановка всех сервисов
- `QUICK_START.md` - краткая справка
